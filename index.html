Data Processing Project

This project demonstrates a Python script that processes sales data, calculates key metrics, and publishes the results as a JSON file via GitHub Pages.

Project Components

The following files are part of this project. You can download them using the links below to set up your repository.

  • execute.py: The Python script that performs data processing. Download
  • data.csv: The input sales data in CSV format. Download
  • .github/workflows/ci.yml: GitHub Actions workflow for continuous integration and deployment. Download

Generated Output

The execute.py script generates a result.json file containing computed metrics. This file is automatically published to GitHub Pages with each successful push to the main branch.
